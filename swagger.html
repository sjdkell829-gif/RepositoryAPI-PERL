GUERREROS

GET /cgi-bin/api.pl — Listar / Filtrar Obtiene todos los guerreros o filtra por parámetros ▼
Parámetros de Query (todos opcionales)
NombreUbicaciónTipoRequeridoDescripción
id query integer opcional Filtra por ID exacto del guerrero
universo query string opcional Filtra por universo (ej: Dragon Ball, Naruto). Case-insensitive.
poder_min query integer opcional Nivel de poder mínimo (devuelve guerreros con poder >= valor)
Respuestas
200 Array JSON con los guerreros que coinciden con los filtros
[ { "id": 1, "nombre": "Goku", "edad": 37, "nivel_poder": 9000, "universo": "Dragon Ball", "rango": "Adulto" } ]
Probar Endpoint
?id= universo= poder_min=

GET /cgi-bin/api.pl/health — Health Check Estado del servidor y permisos del sistema de archivos ▼
Descripción

Endpoint dedicado para monitorear el estado operativo del servidor. Verifica que el archivo datos.json tenga permisos de escritura y que el servicio esté activo. No requiere autenticación.

Parámetros

Ninguno

Respuestas
200Servidor operativo. Incluye estado del archivo JSON.
500Error de permisos o servicio no disponible.
{ "status": "OK", "servidor": "Apache/Perl CGI", "db_writable": true, "timestamp": "2025-01-01T00:00:00Z" }
