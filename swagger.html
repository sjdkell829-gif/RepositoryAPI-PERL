Anime Power API v8.0.0

API RESTful en Perl CGI para gestión de personajes de anime. Soporta operaciones CRUD completas con persistencia en JSON.

Base URL: https://anime-power-api.onrender.com

Requerido en POST, PUT y DELETE. GET es libre.
Guerreros
GET /cgi-bin/api.pl Listar todos o filtrar por parámetros ▼

Parámetros (todos opcionales)

NombreEnTipoRequeridoDescripción
idqueryintegernoFiltra por ID exacto
universoquerystringnoFiltra por universo. Case-insensitive (ej: Dragon Ball)
poder_minqueryintegernoNivel de poder mínimo. Devuelve guerreros con poder ≥ valor

Respuestas

200
Array JSON con guerreros filtrados (o todos si no hay filtros)
[ { "id": 1, "nombre": "Goku", "edad": 37, "nivel_poder": 900, "universo": "Dragon Ball", "rango": "Adulto" } ]

POST /cgi-bin/api.pl Registrar nuevo guerrero 🔒 ▼

Headers

NombreEnTipoRequeridoDescripción
X-Auth-TokenheaderstringsíToken de autenticación. Valor: admin123
Content-Typeheaderstringsíapplication/json

Request Body (application/json)

{
  "nombre": string★ requerido
  "edad": integer (0-999)★ requerido
  "nivel_poder": integer (0-999)★ requerido
  "universo": stringopcional
}

El campo rango se asigna automáticamente: edad < 18 → Joven · 18–99 → Adulto · ≥ 100 → Legendario

Respuestas

201
Guerrero creado exitosamente
{ "status": "Creado", "id": 5 }
401
Token inválido o ausente
